Background

The ImageCapture Web API allows web developers to leverage built-in support for automatic continuous framing based on the position of human faces within the field of view.

Note: Face Framing is still under development. You must be using Chrome with the --enable-blink-features=MediaCaptureCameraControls flag enabled.
